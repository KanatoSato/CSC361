The purpose of a program traceroute.py is to analyze a trace of IP datagram.
The program captures the source and the ultimate destination nodes as well as the intermediate nodes.

1. Unzip my .tar file
2. Place pcap file in the smae directory as unzipped files.
3. Run traceroute.py as follows:
    python3 traceroute.py traceroute-frag.pcap
